Constantinople

‎ La splendide ville ouvrait ses larges côtes aux bras du matin. Les longs rayons glissaient comme des navires dans le port. Les dorures s’émerveillaient du zénith. C’était le feu à toutes les rues ! Les tentures damassées demandaient à leurs amies le chemin de l’horizon. Un doux soir – doux de toutes les joies guidées –, les palmiers s’allongèrent. Leur long cou de girafe riait dans les eaux tranquilles ; Les mâts : « Je veux respirer la saison ! » Les coqs s’endormaient les derniers. L’automne s’excusa. Il n’y a pour faire dorer les joues du mariage que les obscurs soleils entre les feuilles. Et le calme clair de lune souriait derrière les nuages obliques et inconnus.
